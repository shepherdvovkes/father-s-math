Инструкции:

  1. Нажмите на "Главная заставка" для просмотра полной анимации
  2. Дождитесь завершения анимации (6 секунд)
  3. Появится кнопка "Перейти к приложению"
  4. Нажмите кнопку для перехода к основному приложению
  5. При обновлении страницы (F5) заставка запустится заново
